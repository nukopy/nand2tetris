## Chapter 01 - ブール論理

NAND をプリミティブゲート（論理回路の最小構成要素）として，種々の論理回路を構築する．

### 用語

- ブール値（バイナリ値，2 値，論理値） boolean
  2 状態を表す値．「0/1」，「True/False」，「ON/OFF」，「Yes/No」などがラベルとして用いられる．
- ブール代数 boolean algebra
  ブール値を代数的に扱う学問分野．
- 論理演算 logical operation
  ブール値の演算．
- ブール関数（論理関数） bool(logical) function
  入出力がブール値である関数．
- 論理回路 logic circuit
  論理演算を行う（デジタルな）電子回路．状態を持たない（入力で出力が定まる）「組合わせ回路」，状態を持つことのできる「順序回路」の 2 つに大別される．論理回路の設計には論理式，真理値表，回路図が用いられる．回路図の表記手段として MIL 記号が主に用いられる．
- 論理ゲート logic gate
  ブール関数を実現するための物理デバイス．**論理回路の構成要素**．ダイオード，トランジスタ等を用いて実装される．基本的な論理演算（AND，OR，NOT）を行う論理ゲートのことを基本論理ゲートという．
- 論理設計 logic design
  論理回路の設計のこと．論理設計の基本的な要求は，**特定のインターフェース（仕様）を何らかの方法で実現すること**である．効率の点では，「do more less」，つまり「出来る限り論理ゲートの数を減らす」ことが一般的な指標となる．つまり，論理設計の目的は，論理ゲート（or 論理回路）の仕様（インターフェース）が与えられたとき，すでに実装済みの他の論理ゲートを用いて対象の論理ゲートを実装する効率的な方法を探すことである．
- ハードウェア記述言語（Hardware Description Language，HDL）
  （物理的な実装ではなく）コンピュータ上で論理回路を設計・実装するためのプログラミング言語．
- ハードウェアシミュレータ hardware simulator
  HDL で書かれたプログラムをテストすることができるソフトウェア．設計した論理回路が要求を満たすか，入力に対して期待される出力結果が得られるかをテストできる．HDL で書かれたプログラムを読み込み，メモリ上に指定された論理回路を構成し入出力をシミュレートする．

### プリミティブゲート

- NAND: NOT AND．論理積の否定．

```hdl
CHIP Nand {
    IN a, b;
    OUT out;

    PARTS:
    /*
    if a = b = 1:
        then out = 0;
    else out = 1;
    */
}
```

補数を簡単に求めるには，x をビット反転させ，(1)_2 を足せば良い．

### HDL メモ

#### 出力先は複数書ける

```java
And16(a=x, b=y, out[0..7]=aout, out[8..15]=bout, out=out, out=anotherOut);
```

複数ビットを出力する場合は部分的に別々に出力したり，別名を付けたりできる．**特に，内部ピンの接続時に一方は出力させたいが別の入力にも使いたい**というときに役立つ．

#### 内部バスはスライスできない

複数ビットを出力する書き方をすると，勝手に複数ビットに対応した内部ピン（内部バス）ができる．
この内部バスはスライス表記ができない．

- input として定義したバスはスライス表記ができる

```java
// inputとして定義したバスはスライス表記ができる
CHIP Or16Way {
  IN
    in[16];
  OUT
    out;
  PARTS:
  Or8Way(in=in[0..7], out=aout);
  Or8Way(in=in[8..15], out=bout);
  Or(a=aout, b=bout, out=out);
}
```

- ERROR: 自分で定義した暗黙の内部バス（input に書かれていないバス）はスライスできない

```java
CHIP NotOr16Way {
  IN
    in[16];
  OUT
    out;
  PARTS:
  Not16(in=in, out=notin);
  Or8Way(in=notin[0..7], out=aout); //←エラー！
  Or8Way(in=notin[8..15], out=bout);
  Or(a=aout, b=bout, out=out);
}
```

- 回避方法

出力時に別名をつけ，複数出力するようにする．

```java
CHIP NotOr16Way {
  IN
    in[16];
  OUT
    out;
  PARTS:
  Not16(in=in, out[0..7]=notin1, out[8..15]=notin2, out=outNot16);
  Or8Way(in=notin1, out=aout);
  Or8Way(in=notin2, out=bout);
  Or(a=aout, b=bout, out=out);
}
```
